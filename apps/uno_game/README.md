## 游戏规则

### 基本玩法

1. **开局**：每位玩家获得7张牌
2. **出牌**：出与弃牌堆顶牌颜色或数字相同的牌
3. **抽牌**：无法出牌时点击抽牌堆
4. **获胜**：第一个出完所有手牌的玩家获胜

### 特殊卡牌

- **Skip (⊗)**：跳过对手回合
- **Reverse (↻)**：反转游戏方向
- **Draw 2 (+2)**：对手抽2张牌并跳过回合
- **Wild (W)**：可以改变当前颜色
- **Wild Draw 4 (+4)**：对手抽4张牌，可改变颜色

### UNO 规则

当你仅剩1张牌时，必须喊"UNO"！点击屏幕上的UNO按钮即可。

## 如何开始

### 方法 1：本地运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 方法 2：部署到 GitHub Pages

1. 将代码上传到GitHub仓库
2. 在仓库设置中启用 GitHub Pages
3. 访问你的游戏网址
4. 在手机上打开即可游玩

### 方法 3：本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server -p 8000
```

然后访问 `http://localhost:8000`

## 在手机上安装

### iOS (Safari)
1. 访问游戏网址
2. 点击分享按钮
3. 选择"添加到主屏幕"
4. 像原生应用一样使用

### Android (Chrome)
1. 访问游戏网址
2. 点击菜单
3. 选择"安装应用"或"添加到主屏幕"
4. 从主屏幕启动

## 开发说明

### AI 策略

AI使用以下策略选择出牌：
1. 优先出 Wild Draw 4
2. 其次出 Draw 2
3. 然后出 Skip 和 Reverse
4. 最后出普通数字卡
5. 选择颜色时优先选择手中最多的颜色

### 游戏流程

1. 初始化游戏 → 洗牌 → 发牌
2. 玩家回合 → 出牌或抽牌
3. AI回合 → 智能决策
4. 检查胜负条件
5. 循环直到游戏结束
